# 用户故事模板

使用此模板编写包含验收标准三步法的单条用户故事。

## 出处
改编自 `https://github.com/deanpeters/product-manager-prompts` 仓库中的 `prompts/user-story-prompt-template.md`。

## 模板
```markdown
### 用户故事 [ID]：

- **摘要：** [简洁、以用户价值为中心的标题]

#### 用例（用户故事三段式）：
- **作为** [用户名（如有），否则persona，否则角色]
- **我想要** [用户为实现结果而采取的行动]
- **以便** [用户期望的结果]

#### 验收标准（验收标准三步法）：
- **场景：** [描述价值的简洁、可读场景]
- **假如：** [初始上下文或前提条件]
- **并且假如：** [额外的上下文或前提条件]
- **并且假如：** [按需添加更多上下文]
- **并且假如：** [UI聚焦的上下文，确保"当"可以触发]
- **并且假如：** [结果聚焦的上下文，确保"那么"能够交付]
- **当：** [触发行动的事件]
- **那么：** [与"以便"对齐的期望结果]
```

## 注意事项
- 只用一个 **当** 和一个 **那么**。多个当/那么通常意味着故事应该拆分。
- 如需多个结果，请使用故事拆分方法进行分解。
